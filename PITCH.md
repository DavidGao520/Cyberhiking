# Cyber Hiking USA - 1分钟产品介绍

## 🎯 核心定位（10秒）
**Cyber Hiking USA** 是一款基于太平洋屋脊步道（PCT）的生存模拟游戏。体验2,650英里的真实徒步挑战，在极端环境中管理资源、做出生死抉择。

## 🎮 核心玩法（20秒）
- **真实资源管理**：健康、能量、食物、水、心理状态 - 每一个决策都关乎生存
- **随机事件系统**：每次前进都会遇到真实场景 - 迷路、野生动物、恶劣天气、装备故障
- **无痕山林（LNT）系统**：你的选择影响环境评分，高分带来更多正面事件
- **补给站系统**：在城镇购买食物和水，规划你的补给策略

## 💡 独特价值（20秒）
- **教育意义**：通过模拟让玩家理解自然的威力和生命的脆弱
- **真实地理**：基于PCT真实路线和海拔数据
- **动态天气**：海拔影响温度，天气影响能量消耗
- **沉浸式体验**：每次点击都有反馈，每个选择都有后果

## 🚀 技术亮点（10秒）
- React + Vite 构建
- 数据驱动的游戏系统
- 响应式设计，支持移动端
- 实时状态反馈系统
