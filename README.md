# Cyber Hiking: Pacific Crest Trail

A survival simulation game based on the Pacific Crest Trail. Experience the harsh reality of long-distance hiking through dynamic weather, resource management, and life-threatening decisions.

## Features

- **Realistic Geography**: Route nodes based on actual PCT landmarks
- **Dynamic Weather System**: Encounter blizzards, thunderstorms, extreme heat, and fog that affect body temperature and movement speed
- **Resource Management**: Carefully allocate food, water, and equipment weight
- **Random Events**: Make critical decisions based on real-world scenarios like getting lost, wildlife encounters, and equipment failures

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Design Philosophy

This game is not meant to encourage reckless adventure, but rather to help players understand the power of nature and the fragility of human life. Through simulation, we hope players will respect nature and value life, participating in outdoor activities scientifically and safely.
